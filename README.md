# Výzkumné směry VŠB-TUO — Prezentace

Slidev prezentace mapující 12 klíčových výzkumných směrů VŠB – Technické univerzity Ostrava.

## Spuštění

```bash
pnpm install
pnpm run dev
```

Prezentace se otevře na `http://localhost:3030`.

## Struktura prezentace

| Slide | Obsah |
|-------|-------|
| 1 | Úvodní stránka |
| 2 | Přehled témat 1–4 (4 dlaždice) |
| 3 | Přehled témat 5–12 (8 dlaždic) |
| 4–15 | Detail každého výzkumného tématu |
| 16 | Poděkování + odkaz na web |

## Soubory

```
slides.md          # zdrojový soubor prezentace
topics.json        # data témat (názvy, anotace, klíčová slova, kontakty)
topic-XX.jpg       # přehledové obrázky (krajina)
topic-XX-p.jpg     # detailní obrázky (portrét)
title.jpg          # pozadí úvodního slidu

*.jpg              # fotografie garantů
```
